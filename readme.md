## Installation

Dillinger requires [python] 3.0+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd flask_dev
source env/bin/activate
pip3 install -r requirements.txt
```

For production environments...

```sh
python3 app.py
```
